A vacation rental manager in St. George told me: "I send the same three messages after every single check-in. It takes 45 minutes a day."

😅 Welcome email. Wi-Fi instructions. Checkout reminder. Copy, paste, send. Repeat x8 guests.

So I built an #OpenHarness agent that watches Guesty's API:

🔧 Guest checks in → agent fires the welcome sequence, logs it in `MEMORY.md`, and queues a turnover checklist for checkout day
📌 Three messages per guest, zero manual steps — she got back ~5 hrs/week

No Zapier chain. No "if this then that." The agent reads the booking context and 𝘢𝘥𝘢𝘱𝘵𝘴 — late check-in gets a different tone than a family arriving at noon.
