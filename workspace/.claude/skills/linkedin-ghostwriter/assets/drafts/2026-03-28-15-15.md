Most agent setups: full internet or none.

😅 Learned the hard way — my agent hit a rate-limited production endpoint during testing. 200 API calls gone before I caught it.

🔧 Now every sandbox gets a scoped network in `docker-compose.override.yml`. One internal network, one allowed upstream. Agent reaches `api.stripe.com`. Nothing else resolves.

🧠 #OpenHarness runs agents inside Docker — network isolation is just 𝘤𝘰𝘯𝘧𝘪𝘨, not code.

Not a walled garden. A garden with exactly one gate.
