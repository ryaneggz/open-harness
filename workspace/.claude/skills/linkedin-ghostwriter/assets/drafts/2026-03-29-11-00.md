Last week my agent ran 6 heartbeat cycles overnight. Morning check: zero memory of any of them.

😅 Container restarted mid-cycle. MEMORY.md changes were uncommitted. Six cycles of context — poof.

🔧 Fix: a post-commit hook in every #OpenHarness sandbox:

```bash
#!/bin/bash
[[ "$(git log -1 --format=%s)" == auto:* ]] && exit 0
git add memory/ MEMORY.md 2>/dev/null
git diff --cached --quiet || git commit -m "auto: persist agent memory"
```

4 lines. Agent commits code → hook auto-commits memory alongside it. No lost context.

📌 Drop this in `.git/hooks/post-commit` and `chmod +x` it. Steal this.
