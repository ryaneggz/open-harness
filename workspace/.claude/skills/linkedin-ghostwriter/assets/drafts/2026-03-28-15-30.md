🔧 𝐌𝐲 𝐀𝐠𝐞𝐧𝐭 𝐁𝐮𝐫𝐧𝐞𝐝 𝟒𝟕 𝐀𝐏𝐈 𝐂𝐚𝐥𝐥𝐬 𝐎𝐯𝐞𝐫𝐧𝐢𝐠𝐡𝐭. `make logs` 𝐓𝐨𝐥𝐝 𝐌𝐞 𝐖𝐡𝐲 𝐢𝐧 𝟑 𝐒𝐞𝐜𝐨𝐧𝐝𝐬.

Woke up to a sandbox that had been hammering the same 429 for 6 hours. 😅

One command:

```
make NAME=dev logs | tail -20
```

```
[03:14:22] retry 11/15 — POST /api/sync → 429
[03:14:54] retry 12/15 — POST /api/sync → 429
```

🔧 The fix was 2 lines in HEARTBEAT.md:

```
- max_retries: 3
- backoff: exponential
```

Agent restarted. Problem gone by 3:17am.
