I pointed 3 agents at the same repo and asked each to fix a different bug.

Agent 1 refactored the auth middleware. Agent 2 updated the same middleware's error handling. Agent 3 touched the shared config both depended on.

😅 The merge conflict was 47 lines of overlapping changes. I stared at it for 10 minutes before realizing — this was my fault, not theirs.

🔧 The fix: `make NAME=auth quickstart`, `make NAME=errors quickstart`, `make NAME=config quickstart` — three isolated sandboxes, three clean branches, zero conflicts.

📌 #OpenHarness NAME= gives each agent its own workspace. Same repo, different containers, independent git state.
