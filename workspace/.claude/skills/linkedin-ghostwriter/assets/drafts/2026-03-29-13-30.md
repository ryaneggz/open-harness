I wasted a Saturday wiring GitHub Actions for a solo project. 😅 Then I added 4 lines to my #OpenHarness HEARTBEAT.md:

```
### Run tests
- cd workspace && npm test
- If failures: fix and re-run
- Log results to MEMORY.md
```

Every 30 minutes the agent wakes, runs the suite, fixes what it can, logs what it can't.

🔧 No YAML. No runners. No billing page.

Steal this. Paste it into your HEARTBEAT.md and set `HEARTBEAT_INTERVAL=1800`.
