Now every #OpenHarness sandbox ships a `.env.example` — 7 vars, zero secrets:

```
NAME=dev
DOCKER=true
HEARTBEAT_INTERVAL=1800
HEARTBEAT_ACTIVE_START=6
HEARTBEAT_ACTIVE_END=23
ANTHROPIC_API_KEY=
GITHUB_TOKEN=
```

🔧 Config vars get defaults. Secret vars stay blank — your `.env` fills them, `.gitignore` guards them.

🧠 One `cp .env.example .env` and you’re configured. No README spelunking, no “what env vars does this need?”
