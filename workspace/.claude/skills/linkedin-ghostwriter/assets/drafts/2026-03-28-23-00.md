My agent needed to build a Dockerfile, tag it, and push to GHCR. 😅 Inside a container.

One flag:

```bash
git clone https://github.com/ryaneggz/open-harness.git && cd open-harness
DOCKER=true make NAME=dev quickstart
```

🔧 The #OpenHarness sandbox mounts your host Docker socket and dynamically matches the GID at boot
🧠 Your agent gets full `docker build`, `docker compose`, `docker push` — 𝘸𝘪𝘵𝘩𝘰𝘶𝘵 breaking out of the sandbox

No privileged mode. No sidecar daemon. Just the socket.
