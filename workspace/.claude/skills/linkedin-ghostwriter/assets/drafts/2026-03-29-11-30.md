Last week my agent committed a `.env` with a live Zoho API key. Caught it in 4 minutes. Still had to rotate the credential.

😅 Agents don't know what's a secret. That's on you.

Now every #OpenHarness sandbox ships with this:

```
.env
.env.*
credentials*.json
*.pem
*.key
docker-compose.override.yml
.secrets/
**/secrets/
```

🔧 8 lines. Covers 𝘦𝘷𝘦𝘳𝘺 pattern I've seen an agent try to commit — API keys, TLS certs, Docker overrides with embedded tokens.

📌 I bake this into the image so no sandbox starts without it.
