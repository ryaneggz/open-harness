My sandbox gets weird after a week. Stale deps, ghost processes, configs I forgot I tweaked. 😅

🔧 The fix:

```
make NAME=dev nuke && make NAME=dev quickstart
```

90 seconds later: clean Debian image, Node 22, Bun, uv, Docker CLI — all provisioned. SOUL.md and MEMORY.md intact on the bind mount.
