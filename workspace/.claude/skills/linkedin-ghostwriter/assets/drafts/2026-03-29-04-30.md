😅 I had 14 shell aliases for managing agent sandboxes. Then I switched laptops and lost all of them.

Now it's one Makefile. 6 targets, one `NAME=` variable:

```
make NAME=dev quickstart    # clone → build → run
make NAME=dev shell         # drop into the sandbox
make NAME=dev logs          # tail agent output
make NAME=dev heartbeat     # start the heartbeat loop
make NAME=dev rebuild       # nuke and rebuild in 90s
make NAME=dev stop          # shut it down
```

🔧 Same commands work for `NAME=research`, `NAME=frontend`, `NAME=api` — all running independently.

🧠 No docs. No flags. `make` tab-complete handles the rest.
