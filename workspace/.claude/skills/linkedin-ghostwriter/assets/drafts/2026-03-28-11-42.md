Every new #OpenHarness sandbox gets these before I touch anything else:

```
alias ss='make NAME=$SANDBOX shell'
alias hb='tail -20 memory/$(date +%Y-%m-%d).md'
alias nuke='make NAME=$SANDBOX destroy && make NAME=$SANDBOX quickstart'
alias ctx='wc -l CLAUDE.md SOUL.md MEMORY.md'
alias logs='docker logs sandbox-$SANDBOX --tail 50'
```

🔧 `ss` drops me in. `hb` checks today's heartbeat log. `nuke` rebuilds in 90 seconds. `ctx` audits my context files before every session. `logs` tails the container.

😅 I typed these manually for weeks. That's ~20 minutes a day I'll never get back.

📌 Bake them into your sandbox `.bashrc` or the provisioning script — they survive `make rebuild`.
