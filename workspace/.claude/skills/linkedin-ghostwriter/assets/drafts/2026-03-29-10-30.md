Asked my agent to containerize a FastAPI service. It ran docker build 11 times.

😅 Wrong base image. Missing libpq-dev. Broken COPY path. Each failure spawned a full build — layers, cache, dangling images — inside the sandbox.

🔧 DOCKER=true in #OpenHarness mounts the Docker socket into the container. Agent builds, fails, retries — all disposable. My host Docker cache? Untouched.

📌 Build 12 passed. Clean image. Agent pushed to ghcr.io and moved on.

If that happened on my laptop, I'd have 11 dangling images and a headache.

git clone https://github.com/ryaneggz/open-harness.git && cd open-harness
make NAME=dev DOCKER=true quickstart
