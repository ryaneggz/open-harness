I used to `docker compose exec` into the sandbox, `cd workspace`, then `cat MEMORY.md`, then scroll back to find where the agent left off.

Now:

```
docker compose exec dev-sandbox tail -f workspace/MEMORY.md
```

That's it. Real-time stream of every decision my agent writes down. I keep a tmux pane running this while I work on something else.

🧠 In #OpenHarness, agents write timestamped entries to `MEMORY.md` — what they tried, what failed, what they learned. This command lets you watch it happen live.

📌 Last Tuesday I caught an agent looping on the same failing test 4 cycles in a row — spotted it in the tail output within 90 seconds.
