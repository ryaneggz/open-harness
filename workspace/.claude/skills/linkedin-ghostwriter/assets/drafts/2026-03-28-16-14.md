Tuesday night. Agent hits `match` syntax, Python 3.10 chokes, `uv run` fails.

😅 On my host, this is a 45-minute pyenv rabbit hole. Inside the sandbox?

```bash
uv python install 3.12 && uv run pytest
```

Done. Two commands. Host stays on 3.10, agent gets 3.12, nothing conflicts.

🔧 Each `NAME=` sandbox has its own runtime stack. My `NAME=api` runs 3.12, `NAME=ml` runs 3.11 with torch — same machine, 𝘻𝘦𝘳𝘰 cross-contamination.

📌 `make NAME=api quickstart` → isolated Python in under 3 minutes.
