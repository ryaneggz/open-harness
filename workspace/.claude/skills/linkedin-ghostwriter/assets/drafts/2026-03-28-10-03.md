Coffee first. Then these four lines:

```bash
make NAME=dev shell
cat memory/$(date +%Y-%m-%d).md
git log --oneline -10
cat HEARTBEAT.md
```

😅 I used to SSH in and grep through logs for 20 minutes. Now it's 30 seconds.

🔧 In #OpenHarness, `memory/YYYY-MM-DD.md` tells me what the agent 𝘥𝘪𝘥. `git log` tells me what it 𝘤𝘩𝘢𝘯𝘨𝘦𝘥. `HEARTBEAT.md` tells me what's 𝘯𝘦𝘹𝘵.

🧠 Three files. Three questions answered. Zero log spelunking.
