Just kidding. The second agent couldn't start:

`Error: container name "open-harness-dev" is already in use`

Five agents. Same default name. Four of them dead on arrival.

The fix took 30 seconds:

```
make NAME=agent1 quickstart
make NAME=agent2 quickstart
make NAME=agent3 quickstart
```

Each gets its own container, workspace, and git state. No collisions.

😅 The oldest problem in CS — naming things — applies to containers too.
