2am sync run. Agent pulling Square transactions into QuickBooks. 47 line items in, the Zoho webhook fires, rate limit hits — `429 Too Many Requests`. Agent dies.

😅 Old me: grep the logs, figure out which invoices posted, replay from scratch.

🧠 #OpenHarness me: opened `memory/2026-03-25.md` and found this:

```
## Square → QB sync
- Processed: invoices 1-47 of 83
- Failed at: invoice #48, 429 from Zoho webhook
- Pending: invoices 48-83
- QB batch ID: TXN-20260325-0200
```

Restarted from invoice 48. Done in 90 seconds.

📌 MEMORY.md isn't a feature — it's your agent's flight recorder. When things go wrong, it's the difference between 𝘳𝘦𝘣𝘶𝘪𝘭𝘥𝘪𝘯𝘨 and 𝘳𝘦𝘴𝘶𝘮𝘪𝘯𝘨.
