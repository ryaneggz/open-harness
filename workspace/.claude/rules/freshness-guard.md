# Freshness Guard

## Compressed files are read-only

Never edit compressed `.md` files directly when a `.original.md` counterpart exists:
- `AGENTS.md`, `SOUL.md`, `TOOLS.md`, `USER.md`, `HEARTBEAT.md`
- `.claude/rules/{api,code-quality,components,git,issue-triage,nextjs,styles,testing}.md`

These are auto-generated by `/compress`. Edit the `.original.md` source, then run `/compress`.

## After editing any .original.md

Run `/compress` to regenerate the compressed version before committing.
Run `/freshness audit` to verify all pairs are in sync.

## Detection

Pre-commit hook blocks commits where one side changed without the other.
CI workflow validates pair freshness on every PR touching workspace `.md` files.
