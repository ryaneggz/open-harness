# Wiki

This wiki is a persistent, LLM-maintained knowledge base. Pages are generated
by the agent from source documents using the `/wiki-ingest` skill.

## Getting Started

1. Drop source documents into `workspace/wiki/sources/`
2. Run `/wiki-ingest` to process them into wiki pages
3. Use `/wiki-query` to search and synthesize answers
4. Run `/wiki-lint` to health-check the wiki

Pages will appear here as they are ingested.
